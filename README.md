## How To Run

Compile any file with a C++ compiler:

```bash
g++ 03_Basic_Maths/gcd.cpp -o gcd
./gcd
```

On Windows PowerShell:

```powershell
g++ .\03_Basic_Maths\gcd.cpp -o gcd.exe
.\gcd.exe
```

## Progress Philosophy

Each solution is kept in a separate file so that every problem is easy to find, read, compile, and revise. I focus on writing simple code first, then improving readability while preserving the original logic.
